Arbitrary Precision
Calculator

A C-based unlimited digit calculator that performs large number arithmetic using Doubly Linked Lists to overcome native data type overflow limitations.

Source Code Watch Demo

Core Concept

Unlimited Digits

Performs arithmetic beyond int, long, or long long limits.

Doubly Linked List

Each digit stored in a node for flexible dynamic operations.

Manual Arithmetic Logic

Implements addition, subtraction, multiplication at digit level.

Key Features

  • Dynamic memory allocation using malloc
  • Carry and borrow management
  • Digit-wise traversal from head and tail
  • Efficient node insertion and deletion
  • Modular structured implementation
